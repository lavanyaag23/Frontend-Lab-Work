### 🚀 Frontend Lab Projects

This repository contains multiple frontend practice projects built using HTML, CSS, JavaScript, and basic web development tools. It serves as a collection of mini projects developed for learning, experimentation, and portfolio building.

### 📁 Projects Included
### 🧮 Calculator UI

A simple calculator interface built using HTML, CSS, and JavaScript with basic arithmetic operations.

### 💬 ChatBot

A basic chatbot UI demonstrating DOM manipulation and interactive responses.

### 🌐 First Web Page

My first static web page created to understand HTML structure and styling.

### 🔐 Password Strength Checker

A modern password validation tool that checks strength based on:

Length
Uppercase & lowercase letters
Numbers
Special characters
Includes a dynamic strength meter and UI feedback.

A React/JS practice project demonstrating component properties and UI structuring.

### 🛒 Grocery AI App

A mini web app concept for grocery management with AI-based features (UI prototype).

### 🗳 Voting App

A simple voting system interface demonstrating user interaction logic.
